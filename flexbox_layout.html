    < justify-content: space-between 속성으로 자식요소를 일정한 간격으로 정렬 >
    justify-content 속성은 주축을 기준으로 flex item을 수평으로 정렬함.
    다음과 같은 5개의 속성 값으로 다양한 수평 정렬 레이아웃을 만들 수 있음.

    - flex-start(기본값) : 주축의 시작 부분을 기준으로 flex item을 정렬함
    - center : 주측의 중앙을 기준으로 flex item을 정렬함
    - flex-end : 주축의 끝부분을 기준으로 flex item을 정렬함
    - space-around : 주축을 기준으로 flex item을 일정한 간격으로 정렬함
    - space-between : 첫 번째와 마지막 flex item은 주축의 시작 부분과 끝부분에 정렬하고 나머지 flex item을 일정한 간격으로 정렬함.

    < align-items: center 속성으로 자식 요소를 교차축의 중앙에 정렬 >

    align-items 속성은 주축을 기준으로 flex item을 수직으로 정렬함
    - stretch(기본값): flex item의 높으를 늘려 flex container의 전체 높이를 채움
    - flex-start: 교차축의 시작 부분을 기준으로 flex item을 정렬함
    - center : 교차축의 중앙을 기준으로 flex item을 정렬함
    - baseline : 글꼴의 기준선인 baseline을 기준으로 flex item을 정렬함
    - flex-end : 교차축의 끝부분을 기준으로 flex item을 정렬함
